# Saiket Python Internship Projects

This repository contains Python projects completed as part of the **Saiket Systems Internship Program**.  
These projects are designed for beginners to strengthen core Python concepts, logic building, and problem-solving skills.

##  Projects Included

### 1) To-Do List Application
A simple task management application that allows users to add, view, and remove tasks.
**Concepts Used:** Lists, Loops, Functions, File Handling (optional)

### 2) Guess The Number Game
A fun game where the user guesses a randomly generated number.
**Concepts Used:** Random Module, Loops, Conditional Statements

### 3) Basic File Handling
A program demonstrating reading, writing, and appending data to files.
**Concepts Used:** File I/O Operations in Python

### 4) Basic Web Scanner
A basic web content fetcher that reads data from a given URL.
**Concepts Used:** Requests Library, Web Data Handling

### 5) Currency Converter
A simple currency conversion program based on user input.
**Concepts Used:** Functions, Conditional Logic, User Input Handling

##  Technologies Used

- Python 3
- Standard Python Libraries

## Internship Objective

- Strengthen Python fundamentals  
- Understand real-world mini project development  
- Improve logical thinking and coding structure
